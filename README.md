# Cloud Community Club Static Website

This project is a responsive static website developed for the Cloud Community Club using HTML, CSS, and JavaScript. It serves as an informational landing platform to showcase the club’s activities, events, team members, and engagement opportunities.

## Features

- Responsive multi-section layout
- Clean navigation structure
- About section describing the community
- Events section highlighting activities
- Team section showcasing members
- Changelog (timeline) section
- Contact section for engagement
- Modular project structure using separate HTML, CSS, JavaScript, and assets

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Netlify (Deployment)
- GitHub (Version Control)
